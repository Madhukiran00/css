Combinators:

Descendent:-

Targets all child elements nested inside a parent, regardless of how deep.

div p {
    color: blue;
  }






Child:-


Selects elements that are direct children of a specified element.


div > p {
    color: red;
  }
  








Adjacent Sibling:-


Selects an element that is the next sibling immediately after another element.


h2 + p {
    font-style: italic;
  }
  








General Sibling 


Selects all siblings of an element that come after it.


h2 ~ p {
    color: green;
  }
